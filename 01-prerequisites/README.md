# Prerequisites

These are the prerequisites that are needed for the course

## HTML

You should know how to structure a web page using HTML.  
Specifically you should know the following HTML tags

- HTML Document structure

```HTML
<html>
	<head>...</head>
	<body>...</body>
</html>
```

- Typography

```HTML
<h1>
<h2>
<h3>
<h4>
<h5>
<h6>
<p>
<span>
```

- Links

```HTML
<a href="...">
```

- Layout

```HTML
<div>
```

- Lists

```HTML
<ul>
<li>
```
